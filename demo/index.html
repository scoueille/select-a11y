select-a11y - demo

Filter component: multi select transformed to suggestions list with search input. WCAG and RGAA compliant.

select-a11y transforms multi select into suggestions list with search input. It is compliant with Web Content Accessibility Guidelines (WCAG) and General Accessibility Framework for Administrations (Référentiel général d'accessibilité pour les administrations - RGAA).

Read more about why and how [fr].

As seen in this second example, one or more option may have already an attribute selected in source.

To be transformed by select-a11y.js, the select element:

  • must contain the attribute data-select-a11y;
  • must contain an attribute data-placeholder="Text of the placeholder";
  • must contain a class attribute with any value.

select-a11y is part of the PiDILA’s accessible and responsive UI components library Scampi. It was primarily developed for service-public.fr : see filter boxes on this page.
